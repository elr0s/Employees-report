Задача
В CSV-файле содержатся данные по анкетированию сотрудников компании.
Запись о каждом сотруднике - на новой строке. Сотрудников - несколько сотен.
Формат: "Код сотрудника | Оценка навыка1 | Оценка навыка 2"
Оценок - несколько десятков. Каждые три навыка  подряд группируются в группу навыков
Например, 
"навык 1 + навык2 + навык3" = "группа навыков 1" 
"навык 4 + навык5 + навык6" = "группа навыков 2"

Нужно обработать этот файл и построить для каждого сотрудника отчет в следующем формате
Код сотрудника
Номер группы навыков; среднее арифметическое группы навыков сотрудника; дельта между суммой навыков группы участника и медианой в группе навыков всех участников
